# Mausoleum 2.2: Lanchyn Demo

Open in Godot 4.4 and run `res://scenes/main/main.tscn`.

Controls:
- Move: arrows or WASD (Godot defaults: arrows guaranteed)
- Attack: left mouse button or J

Features in this demo:
- 3 selectable heroes
- 3 enemy variants
- 12-wave mode
- Endless mode
- Wall defense + game over on wall/hero death
- Uses provided artwork as hero select/menu/background references
